Cela décrit l'utilisation de l'application. Pour savoir comment créer un calendrier, consultez la documentation en ligne (readme) sur Github.

Ajouter un calendrier

Attention : N'utilisez que des calendriers de personnes en qui vous avez confiance : ouvrir des liens vers des sites malveillants représente un risque pour la sécurité de votre smartphone!

Cliquez sur le bouton "+" et entrez le chemin d'accès complet au fichier de calendrier (par exemple /sdcard/Downloads/MyCalendar.txt) ou ouvrez directement un fichier de calendrier et choisissez "Ajouter un calendrier". Vous pouvez également entrer un titre personnalisé pour le calendrier à afficher dans la liste des calendriers et le titre des notifications.

Visualiser un calendrier

Sélectionnez un calendrier dans la liste pour afficher les détails du calendrier, les paramètres et une liste de ses entrées.
  • Pour chaque entrée, l'heure programmée par le calendrier est affichée à gauche. Si le temps de notification diffère de celui de la première, il est indiqué entre parenthèses. A côté de l'heure, la description de l'entrée est affichée. Selon si le paramètre "Afficher les entrées" est coché ou pas, la description peut être masquée.
  • Cliquez sur une entrée pour afficher sa description et ses options, pour voir et ouvrir le lien.

Notifications

Une notification est affichée exactement une fois pour chaque entrée dans chaque calendrier. Sauf indication contraire, elle est indiquée exactement à l'heure telle que définie par le calendrier. Lors de l'ajout d'un calendrier ou de l'activation des notifications, les notifications des entrées déjà passées sont également affichées.

Paramètres du calendrier

  • Activer les notifications : si cette option est cochée, les notifications s'affichent. Lors de la réactivation, les notifications des entrées déjà passées seront affichées.
  • Délai de notification le plus court : si cette option est activée, aucune notification ne sera affichée avant l'heure indiquée - si elles sont programmées plus tôt dans le calendrier, elles seront affichées exactement à l'heure spécifiée ici. Si l'auteur du calendrier n'a pas spécifié d'heure, alors les entrées sont programmées pour 0:00 - vous pouvez ainsi vous assurer que vous ne recevrez pas de notification quand vous ne voulez pas en recevoir.
  • Afficher les entrées : permet de masquer toutes les entrées ou les entrées futures, de ne pas révéler le texte de description à l'avance. Les entrées peuvent être visualisées en cliquant dessus.
